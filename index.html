Hi, I'm Pieter Willekens.

I'm a developer driven by curiosity about technology. The field's constant evolution keeps me engaged and motivated to explore new possibilities.

At work

At Interactive Studios, I serve as tech lead for the Patient Journey App, a health-focused application that provides patients with contextual information at critical moments. I manage both the technical development and infrastructure, handling everything from application architecture to CI/CD optimization and DevOps work using Kubernetes.

Technologies

My technical work spans backend development with Laravel and PHP, frontend development using React and TypeScript, and mobile development with React Native and Expo. I work extensively with Kubernetes for deployment orchestration and am exploring Rust for its performance characteristics. I actively contribute to open source projects and aim to increase my involvement in the community.
